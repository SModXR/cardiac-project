Acerca del Proyecto

Este proyecto es un sistema de evaluacion cardiovascular que utiliza tecnologia IoT con dispositivos ESP32 y sensores MAX30102 para capturar y analizar la frecuencia cardiaca en tiempo real. El sistema compara los datos obtenidos durante un minuto de medicion (24 lecturas) contra un conjunto de datos de referencia para proporcionar una evaluacion personalizada.

La aplicacion permite a cualquier persona realizar una evaluacion rapida de su frecuencia cardiaca ingresando sus datos basicos (edad, sexo, peso, altura, rutina fisica) y conectandose al dispositivo ESP32 con sensor optico MAX30102 que captura las pulsaciones en tiempo real mediante fotopletismografia (PPG).

Los resultados se procesan mediante algoritmos que consideran factores como edad, sexo, IMC y nivel de actividad fisica para generar una clasificacion y recomendaciones basadas en patrones de salud cardiovascular.

Tecnologias Utilizadas

Infraestructura IoT y Cloud Computing

ESP32 + MAX30102

Microcontrolador ESP32 con sensor optico MAX30102 que captura las pulsaciones cardiacas mediante fotopletismografia (PPG) y las envia a la nube en tiempo real via WiFi.

Azure IoT Hub

Plataforma de Microsoft Azure que gestiona la comunicacion bidireccional entre dispositivos IoT y servicios en la nube de forma segura y escalable.

Azure Functions

Servicio serverless que procesa los datos de BPM, realiza calculos estadisticos y genera evaluaciones comparando con el dataset de referencia.

Blob Storage

Almacenamiento en la nube de Azure para guardar el dataset de referencia (CSV) con 744 registros de frecuencia cardiaca de diferentes perfiles.

Algoritmos BPM

Sistema de analisis que clasifica la frecuencia cardiaca en Bradicardia (<60 BPM), Normal (60-100 BPM) o Taquicardia (>100 BPM) segun estandares clinicos.

Dataset de Referencia

Base de datos con 744 registros de BPM de 31 perfiles diferentes, considerando edad, sexo, IMC y rutina fisica para realizar comparaciones precisas.
